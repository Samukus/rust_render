Dependencies:

1.  Install rust - [link](https://www.rust-lang.org/tools/install)
2.  Download SDL 2.0 library - [link](https://www.libsdl.org/download-2.0.php)
3.  Build and run:
    * `cargo build`
    * `cargo run`
